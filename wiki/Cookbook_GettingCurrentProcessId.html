Getting the current process id (PID) using "foreign.ss"

Problem

We may want to get the current process id (PID) of the current running process.

Solution

We can use the foreign function library to access C's getpid() function: 

(module get-process-id mzscheme
  (provide get-process-id)
  (require (lib "foreign.ss"))
  (unsafe!)
  (define get-process-id (get-ffi-obj 'getpid (ffi-lib #f) (_fun -> _int))))

Discussion

Another approach might be to write an extension (http://list.cs.brown.edu/pipermail/plt-scheme/2004-February/004499.html), or to use the c-ffi (http://list.cs.brown.edu/pipermail/plt-scheme/2004-February/004510.html).
