DrScheme

DrScheme is a sophisticated graphical programming environment for Scheme.

In addition to unrivalled program analysis tools, DrScheme includes a HelpDesk which provides a comprehensive source of information about PLT Scheme, as well as Scheme in general.

DrScheme runs under Windows, Mac OS X, and Unix/X. It is free software, and can be downloaded from the DrScheme home page.
