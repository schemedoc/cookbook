Deleting a File

Problem

You want to delete a file.

Solution

Use plt scheme's standard function delete-file:

Discussion 

(delete-file path)

deletes the file with pathname path if it exists, returning void if a file was deleted successfully, otherwise the exn:i/o:filesystem exception is raised. If path is a link, the link is deleted rather than the destination of the link.
