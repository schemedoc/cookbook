text-template

Location

http://text-template.sourceforge.net

Description

text-template is a small text template library for scheme. It can be used to generate html, sql or whatever text. Currently it supports SISC and PLT-Scheme scheme implementations, but it could be easily ported to other implementations. This is an open source software licensed under GPL.
