File Manipulation

Introduction

Sometimes you gotta interact with the file system and when you do things can get messy. File systems are dependent upon the operating system and since operating systems will handle things in different ways it can be hard to write code that will work on all platforms. Luckily PLT Scheme provides some functions to do platform independent manipulation of directory and file paths.
