Deleting from a Hash

Problem

Remove an entry (key value) from a hash table.

Solution

hash-table-remove! removes the value mapping for the given key if it exists in hash-table. If hash-table is immutable, an exception will be raised, see recipe HashRecipeImmutable. 

(hash-table-remove! hash-table key-v)
