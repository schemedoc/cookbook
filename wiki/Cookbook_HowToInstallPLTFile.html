How To Install a .plt File

  1. In DrScheme choose "Intall .plt File" in the "File..." menu, select the Web tab, and give it url where the file is on the web, then click OK.

If you're downloading from someplace like SourceForge ?, it may not be obvious what the URL for the .plt file is. In that case, you can do this:

  1. download it to your machine
  2. In DrScheme choose "Intall .plt File" in the "File..." menu, select the File tab, and give it the path to where you downloaded it.
  3. That's it

Note: Use the HelpDesk to find the documentation. Search for the name of the collection.

On machines without DrScheme, you can use a command line interface instead. Give the path of the .plt file to either "Setup PLT" (Windows) or setup-plt (Unix).
