Creating Chapters

Problem

You want to create a new Chapter in the Cookbook

Solution

Creating a new Chapter is not as simple as creating a recipe, as a Chapter consists of a number of pages, some of which you have to create by hand. The recipe for creating a Chapter is as follows:

  • Assume you want to create a chapter called Foo.
  • Visit NewChapter and create a page called FooChapter ?. Ensure the TopicType is set to Chapter.
  • Create the pages FooIntroduction ?, FooRecipes ? and FooChapterComments ? using the templates given in ChapterStructure. Remember to set the TopicType to Section and ParentTopic to FooChapter ? for these pages.

Note that the naming convention given above is important for correct functioning of the site. You will be cursed for eternity if you wilfully ignore it.
