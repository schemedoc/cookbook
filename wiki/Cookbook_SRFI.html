Scheme Request for Implementation - SRFI

The SRFIs define the closest thing the fiercely independent Scheme community has to a standard library. PltScheme supports a large number of SRFIs. Search HelpDesk for srfi to get a full list.

See http://srfi.schemers.org/ for further details on the SRFI process and SRFI libraries
