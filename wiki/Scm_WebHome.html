Schematics Cookbook Source Code Repository

This is the Scm web, where source code files for the Cookbook are stored.

Files stored here can be included in recipes in the Cookbook using the TWiki %INCLUDE% tag - see IncludingSourceCode for more details.

Any piece of code that should be distributed as part of the cookbook's source distribution should be stored here. This makes it much easier to create the source distribution than if source code is included inline on recipe pages. Inline examples should be limited to small examples or snippets that would be of little use to someone working with the Cookbook code.
