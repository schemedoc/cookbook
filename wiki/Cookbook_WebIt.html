WebIt!

WebIt! is an XML framework for Scheme. WebIt! supports the creation of XML, HTML, and CSS using Scheme. It provides a powerful "macro system" for transformation of XML.

For more information, see the WebIt! Home Page

Installation

* Install the WebIt collection from The WebIt home page

* Install ssax and sxml from Lisovsky's PLT ports

* Install the ssax-rsxml collection from The WebIt home page

* Install the sxpath-webit collection from The WebIt home page
