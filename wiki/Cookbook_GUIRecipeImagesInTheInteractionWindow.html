Displaying an Image in the DrScheme Interaction Window

Problem

Displaying an image in the DrScheme Interaction Window.

Solution

The interaction window can display various non-text related entities called snips. Image-snips are used to display images. 
; Entered in REPL
>(make-object image-snip% "pacman.gif")
[image of PacMan]

Discussion

The solution loads the image "pacman.gif" from disk, instantiates an object of the class image-snip%. When the object is returned to the interaction window the image is automatically displayed.

The supported file formats are: gif gif/mask jpeg png png/mask xbm xpm bmp pict.
