Adding an Element to a Hash

Problem

You need to add an entry to a Hash.

Solution

Simply assign the element to the hash using hash-table-put!. 

(let ((t (make-hash-table)))
    (hash-table-put! t 1 "foo")
    (hash-table-put! t 2 "bar"))

Note however that you have to pass the key for this element as the second argument.

Discussion

Note that hash-table-put! is destructive, so if you pass the same key a second time, the new element will overwrite the old one associated to the key.
